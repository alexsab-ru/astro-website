# SEO

### Входные параметры

```js
interface Props {
  mdx: string;
  noPadding?: boolean;
}
```

### Использование
```js
---
import SeoText from '@/components/Seo/Index.astro';
---
```

```astro
<SeoText mdx="demo-seo" noPadding={false} /> 
```

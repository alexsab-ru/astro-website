# Timer

### Входные параметры

```js
interface Props {
	title?: string;
	subtitle?: string;
	endtime?: Date;
	btnName?: string;
	class?: string;
}

const { 
	title, 
	subtitle, 
	endtime = '2025-12-31T23:59:59+04:00', 
	btnName, 
	class: className,
	...rest 
} = Astro.props;
```
### Использование
```js
---
import Marquee from '@/components/Marquee/Index.astro';
import { LAST_DAY, MONTH, YEAR } from '@/js/utils/date';
---

<Timer 
    title='До конца акции осталось'
    subtitle='' 
    endtime=`${YEAR}-${MONTH}-${LAST_DAY}T23:59:59+04:00` 
    btnName='Зафиксировать цену'
    show={false}
/>

```

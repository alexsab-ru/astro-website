# Marquee

### Входные параметры

```js
interface Props {
	marquee: TMarquee;
	classes?: string;
}

type TMarquee = {
	title: string;
	count?: number;
	speed?: number;
	show: boolean;
}
```
### Использование
```js
---
import Marquee from '@/components/Marquee/Index.astro';
import { LAST_DAY, MONTH, YEAR } from '@/js/utils/date';
---

<Marquee 
    marquee={{
        title: `Тотальная распродажа до ${LAST_DAY}.${MONTH}.${YEAR}`, 
        count: 8,
        speed: 20, 
        show: true
    }}
    classes="relative my-10"
/>
```
### TMarquee
<div class="*:!text-left mb-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
	|title|-|**Обязательный параметр**. Возможность передать HTML теги.|
	|count|8|`number` Количество элементов|
	|speed|20|`number` Скорость анимации в секундах|
	|show|`false`|`true` or `false` Показ|
</div>

<div class="*:!text-left mb-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
	|classes|`absolute w-full left-0 z-10 top-[50vh] md:top-[70vh]`|Классы бегущей строки|
</div>

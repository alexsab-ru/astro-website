# Model

#### Входные параметры

```js
interface Props {
	model: any;
	breadcrumb?: TBreadcrumb[];
}
```

#### Использование
```js
---
import Model from '@/components/Model/Model.astro';
import modelsData from '@/data/models.json';
const model = modelsData.models.filter(model => model.show)[1];
const breadcrumb = { name: `Модели`, href: "/models/" };
---
```

```astro
<Model 
  model={model} 
  breadcrumb={[breadcrumb, {name: model.name, href: model.id}]} 
/>
```

<div class="*:!text-left my-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
	  |**model**|*|Объект модели. **Обязательный параметр**|
    |breadcrumb|-|`TBreadcrumb[]` массив объектов хлебных крошек|
</div>

# Contacts

### Входные параметры

```js
interface Props {
	mapImg?: string;
	title?: string;
    height?: number;
	yOffset?: number;
}
```

### Использование
```js
---
import Contacts from '@/components/Contacts.astro';
---

<Contacts title="Наши контакты" mapImg="https://cdn.alexsab.ru/maps/map-bg.webp" height="500" yOffset="20" />

```

<div class="*:!text-left my-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
	|mapImg|`https://cdn.alexsab.ru/maps/map-bg.webp`|Фоновое изображение, показывается до полной загрузки карты|
    |title|`Наши контакты`|Заголовок секции. Если заголовок не нужен, можно передать пустое значение `title=""`|
    |height|`500`|Высота карты (суммируется с параметром `yOffset * 2`)|
    |yOffset|`20`|Паддинги по горизонтали|
</div>

Внутри компонента `Contacts` используются 2 компонента - `MapInfo` и `Map`. Оба компонента принимают массив `salons`, который берется из `src/data/salons.json`, и строку `classes`.

#### Входные параметры

```js
type TButtonData = {
	view?: string,
	classes?: string,
	url?: string,
	title?: string,
	dataTitle?: string,
	dataFormName?: string,
	icon?: string,
	iconClasses?: string,
  ...rest?: Attr
}
```

#### Использование
```js
---
import Button from '@/components/Button/Button.astro';
---
```

```astro
<Button 
  view="popup-link" 
  classes="btn flex flex-row-reverse items-center gap-3" 
  url="#входные-параметры" 
  title="Button with icon" 
  dataTitle="Title for modal"
  dataFormName="Form name"
  icon="mdi:alert-circle-outline" 
  iconClasses="text-2xl" 
  disabled
/>
```

<div class="*:!text-left mb-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
    |view|`link`|Тип (вид) кнопки `link`, `form-button`, `popup-link`, `scroll-link`|
    |classes|-|классы кнопки|
    |url|-|ссылка, если тип - `link`, `popup-link` or `scroll-link`|
    |title|`BUTTON`|имя кнопки|
    |dataTitle|-|Заголовок модального окна, если тип - `popup-link`|
    |dataFormName|-|Имя формы, если тип - `popup-link`|
    |icon|-|Иконка. Задается имя [Material Design Icons](https://pictogrammers.com/library/mdi/), например, `mdi:car-key`, либо имя svg иконки, расположенной в `src/icons/`|
    |iconClasses|-|классы для иконки|
    |...rest|-|любые атрибуты, например `disabled`|
</div>

### **Input Field**

<div class="max-w-xl mb-10">
  <Input
    type="tel"
    name="phone"
    classes="border-gray-100"
    placeholder="+7 (999) 999-99-99*"
    required
    errorText="Введите корректный номер телефона"
    errorClasses="text-red-500 !block"
  />
</div>

#### Входные параметры

```js
interface Props {
    type?: string,
    name?: string,
    classes?: string,
    placeholder?: string,
    required?: boolean,
    errorText?: string,
    errorClasses?: string,
}
```

#### Использование
```js
---
import Input from '@/components/Input.astro';
---
```

```astro
<Input
  type="tel"
  name="phone"
  classes="border-gray-100"
  placeholder="+7 (999) 999-99-99*"
  required
  errorText="Введите корректный номер телефона"
  errorClasses="text-red-500"
/>
```

<div class="*:!text-left mb-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
    |type|`text`|стандартный атрибут `type`|
    |name|-|стандартный атрибут `name`|
    |classes|-|CSS классы для поля ввода|
    |placeholder|`Введите значение`|стандартный атрибут `placeholder`|
    |required|`false`|обязательное поле?|
    |errorText|-|Текст ошибки обязательного поля|
    |errorClasses|-|CSS классы ошибки, например `text-red-500`|
</div>

# Callback

### Входные параметры

```js
interface Props {
    bgUrl?: string;
    title?: string;
    subtitle?: string;
    formName?: string;
    btn?: string;
}
```

### Использование
```js
---
import Callback from '@/components/Callback.astro';
---

<Callback />

```

<div class="*:!text-left my-10">
    |**Имя параметра**|**По умолчанию**|**Описание**|
    |--|--|--|
	|bgUrl|`https://cdn.alexsab.ru/models/default-model-bg.webp`|Фоновое изображение секции|
    |title|`Улучшим предложение конкурентов!`|Заголовок секции. Если заголовок не нужен, можно передать пустое значение `title=""`|
    |subtitle|`<p>У Вас уже есть предложение от другого дилера?</p><p>Сделаем лучше!</p><p>Оставьте Ваши контакты и в ближайшее время мы перезвоним!</p>`|Подзаголовок секции. Если не нужен, можно передать пустое значение `subtitle=""`. Можно передать HTML тэги|
    |formName|`Улучшим предложение конкурентов`|Имя формы при отправке|
    |btn|`Отправить`|Текст кнопки в форме|
</div>
